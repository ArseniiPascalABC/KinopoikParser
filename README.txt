Добрый день, перед вами ручная реализация для сбора рецензий о фильме "Побег из Шоушенка".
Что бы код заработал у вас должен быть установлен Google Chrome версии 121(последняя на данный момент),
либо установлен другой драйвер, для вашей версии Google Chrome, ссылка на драйвера, драйвер лежит в корне проекта.
Замените его при необходимости, либо установите chrome версии 121(на 31.01.2024 это последняя версия).
Все что вам нужно сделать после запуска кода, это возможно решить капчу, ибо кинопоиск не пропускает множество запросов,
и начинает накидывать капчу. Капчу нужно решить в течении 30 секунд, иначе код не сможет отработать.
Так же будет вторая реализация, только уже при помощи неоф. API.
Ссылка на Google WebDriver - https://chromedriver.chromium.org/downloads Будьте внимательны, каждый драйвер подходит к
конкретной версии Chrome.
